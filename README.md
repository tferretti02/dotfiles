# dotfiles

Agent skills, installable with [`npx skills`](https://github.com/vercel-labs/skills).

## Install

```
npx skills add tferretti02/dotfiles --all -g
```

Use `--skill <name>` instead of `--all` to install just one.

## Skills

- `atlas` - generate an HTML page with docs and diagrams from a problem.
- `grill-me` - interview me about a plan until we agree on it.
- `ship-it` - review, test, and lint committed changes before they ship.
